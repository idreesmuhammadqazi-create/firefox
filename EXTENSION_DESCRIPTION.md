# PseudoRun Firefox Extension - Store Description

## Primary Description (250 characters max)

Complete IGCSE/A-LEVELS pseudocode editor and simulator that works 100% offline. Write, debug, and execute Cambridge pseudocode with real-time validation. Features 22 built-in examples, interactive tutorial, syntax reference, practice problems, and exam mode. Perfect for Computer Science students learning programming concepts without internet dependency.

1. **Install Extension**: Click "Add to Firefox" or load via about:debugging
2. **Open Editor**: Click the PseudoRun icon in your Firefox toolbar
3. **Start Coding**: Write pseudocode with real-time syntax highlighting
4. **Execute Programs**: Click "Run" to see animated output execution
5. **Learn and Practice**: Access tutorial, examples, and practice problems
6. **Save Work**: Programs auto-save locally for continued learning
